# spam-email-classifier

An explainable spam email classification system that analyzes email content using linguistic and structural patterns to identify spam messages in a transparent and interpretable manner.

The **Spam Email Detector** is designed to classify emails as **Spam** or **Legitimate** based on commonly observed textual patterns rather than opaque predictions. The system focuses on **heuristic-based Natural Language Processing (NLP)** techniques to demonstrate how spam detection works at a foundational level.

1. The user pastes an email message into the input box

   - Urgency indicators  
   - Repetitive or suspicious phrases  
3. A rule-based score is calculated  
4. The email is classified as **Spam** or **Not Spam**

    About the Model

    This system uses a rule-based Natural Language Processing (NLP) approach inspired by common spam patterns found in public SMS and email datasets.

    The classifier evaluates keyword frequency, keyword density, and message length to generate an explainable prediction with a confidence score.

    Model Limitations

    • Keyword-based approach without semantic understanding
    • Language-dependent (English-focused)
    • Cannot detect context-aware or adversarial spam

    Future Improvements

    • Supervised ML models (Naive Bayes, Logistic Regression)
    • Context-aware NLP models
    • Dataset-driven training
